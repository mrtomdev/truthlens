# Changelog

All notable changes to TruthLens will be documented here.
This project follows [Semantic Versioning](https://semver.org/).

## [1.0.0] — 2026-05-13

### Added
- Initial public release.
- Manifest V3 Chromium extension.
- Text detector: burstiness, common-word ratio, phrase fingerprint, repetition,
  syntactic uniformity.
- Image detector: known-AI-host check, filename hints, high-frequency energy,
  color entropy, noise uniformity.
- Audio detector: spectral flatness, silence regularity, pitch jitter.
- Inline badges, floating verdict overlay, detail popover.
- Toolbar popup with per-page and lifetime stats.
- Options page for fine-grained scan control.
- Keyboard shortcut (`Ctrl+Shift+T`) and right-click "Scan with TruthLens".
- Smoke tests for the text detector.
